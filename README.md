## Project Overview
This project involves **end-to-end manual quality assurance testing** of **BDShop**, an e-commerce web application. The objective was to ensure **functional accuracy, usability, and reliability** across key user-facing features before release.

Testing was conducted from a **real end-user perspective**, covering critical workflows such as registration, login, product browsing, cart management, checkout, and order tracking.

---

## Scope of Testing
The following modules and functionalities were tested:

- Header  
- Search  
- Footer  
- Homepage Carousel  
- Registration  
- Login  
- Forgot Password  
- My Profile  
- Profile Update  
- Product Listing  
- Cart  
- Checkout  
- Order History  
- Contact Us  
- About Us  

---

## Types of Testing Performed
- Manual Functional Testing  
- Regression Testing  
- Session-Based Testing  
- Validation Testing  
- UI & Usability Testing  

---

## Key Activities
- Analyzed application workflows and requirements to create test scenarios.
- Designed and executed **detailed test cases** for all major modules.
- Performed **end-to-end testing** of user journeys from authentication to order completion.
- Identified, documented, and reported **functional, UI, and validation defects** with clear reproduction steps.
- Verified **input validations, error handling, session management, and navigation flow**.
- Evaluated the application from a **user experience and usability** perspective.
